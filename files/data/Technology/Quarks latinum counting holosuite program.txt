RECREATION: Quark’s latinum counting holosuite program

The coded holosuite program that Quark used to gloat over his gold pressed latinum.  The program was set in an ancient, dank, mouldy dungeon and contained a fine jailwood table.  Quark kept the latinum in a antique treasure chest.
