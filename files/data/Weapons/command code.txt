TECHNOLOGY: command code

Each Starfleet vessel is equipped weith its own unique combination code to prevent an enemy to take control of the computer system from remote. These codes can be retreived from Starfleet Command if the requesting officer has proper clearance. Admiral James Kirk accessed the command codes of the U.S.S. Reliant and forced the ship to drop its shields and to disarm its weapons after the craft, under the command of Khan Noonien Singh, used the vessel to attack the the U.S.S. Enterprise.

Reference: The Wrath of Khan
